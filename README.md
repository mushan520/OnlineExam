# OnlineExam
在线考试系统，上传题目答案，自动识别，给出评分
本系统包括node，sql数据，老师端，学生端；注意，node端图片识别使用的是百度的一个叫sdk的库，在下载依赖时，先下载依赖，在将sdk包下载并放在modules中
